# Исследование биоинспирированных механизмов внимания (SMA-Attention)

Репозиторий содержит код и результаты исследования модификации механизма внимания (Self-Attention) в архитектуре трансформера на основе алгоритма слизевика (**Slime Mould Algorithm — SMA**).

В классических трансформерах механизм `Softmax` обладает высокой чувствительностью к шуму из-за экспоненциального характера преобразования. В данной работе предложен и реализован подход, заменяющий `Softmax` на биоинспирированный механизм взвешивания, основанный на принципах конкурентного распределения ресурсов в колониях слизевика (*Physarum polycephalum*).

**Ключевые результаты:**

* Разработаны два варианта модификации: **SMA-C** (упрощенный, с обучаемым параметром $\beta$) и **SMA-A** (полный, с логарифмической формулой).
* Экспериментально доказано, что предложенные механизмы обеспечивают повышенную устойчивость к шуму (снижение деградации точности на 16–25% при высоком уровне шума) без потери качества на чистых данных.
* Стабильность модели при наличии шума объясняется ограниченностью весовой функции (в отличие от неограниченного `Softmax`).
